# Introduction

In order to configure NeoVim on your system you have to:

- Install prerequisites

```bash
./install-prerequisites.sh
```

- Run the configuration script for NeoVim

```bash
./config-nvim.sh
```

- Open NeoVim with `nvim` and run `:PlugInstall`
